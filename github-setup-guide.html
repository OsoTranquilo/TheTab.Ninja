Create a GitHub Account

Create GitHub Account

Go to GitHub: Visit github.com
Sign up:
• Click "Sign up" in the top-right corner
• Enter your email address and follow the instructions to create an account
Verify your account:
• Check your email for a verification link and click it

Create a New GitHub Repository

Log in: Log in to GitHub with your new account
Create a new repository:
• Click the "+" in the top-right corner and select "New repository"
• Fill in the following details:
- Repository name: Test4TabNinja (any name goes but we will use this name in the example)
- Description: (optional, e.g., "Repository for TabNinja's datasync")
- Choose whether it should be Public or Private. We will use Private, but public could be used if you want to share the bookmarks with others.
- Check "Add a README file"
• Click "Create repository"

Create a Fine-grained Token

Go to settings:
• Click your profile picture in the top-right corner and select "Settings"
Navigate to tokens:
• In the left-hand menu, go to "Developer settings" > "Personal access tokens" > "Fine-grained tokens"
Generate token:
• Click "Generate new token"
• Select a repository access: Only select the repository you created above (Test4TabNinja)
• Select the previously created repository (e.g., Test4TabNinja)
• Expiration: Set to "No expiration"
• Select repository permissions:
- Actions: Read, Write
- Commit statuses: Read, Write
- Contents: Read, Write
- Custom properties: Read, Write
- Metadata: Read
- Pull requests: Read, Write
• Click "Generate token"
Important: Copy and save the token securely - it will only be shown once!

Create a Codespace

Navigate to your repository:
• Go to the Test4TabNinja repository on GitHub
Start a Codespace:
• Click "Code" > "Codespaces"
• Click "Create Codespace on main"
Configure the Codespace (optional):
• Once the Codespace starts, you can install tools and configure the environment as needed
